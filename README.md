# Daniel Herbert Portfolio

A personal portfolio for Daniel Herbert, a content specialist with experience across retail, education, gaming and entertainment.

The site takes inspiration from Windows XP and early 2000s web design. It includes project summaries, detailed case studies, embedded video work, a showreel, an About Me page and an embedded CV.

## Viewing the site

The files in this repository are a complete static website. Open the published GitHub Pages address in a modern browser to view it.

Some videos are embedded from YouTube and Instagram, so they require an internet connection and may be affected by browser privacy settings or changes made by those platforms.

## Publishing with GitHub Pages

1. Upload all files from the export package to the root of the repository.
2. Open the repository's **Settings** and select **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`, then save.

GitHub will provide the public website address after the deployment finishes.
